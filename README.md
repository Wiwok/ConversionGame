# Conversion Game
A game to learn binary and hexadecimal conversions.

## Getting Started

1. Clone Conversion Game

    ```bash
    git clone https://github.com/Wiwok/ConversionGame ConversionGame
    ```

2. Change into the directory

    ```bash
    cd ConversionGame
    ```
3. Install the requirements
    ```bash
    npm i
    ```
4. Launch the game
    ```bash
    node .
    ```

5. Have fun !
